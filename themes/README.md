## ğŸ“˜ Usage Guide

### Method 1: Manual Application (Recommended)

This is the **recommended approach** for learning and full control:

1. **Choose a theme** from the gallery above
2. **Read the theme's README** for specific guidance
3. **Navigate to snippets**: `themes/<theme-name>/snippets/`
4. **Open a snippet file** (e.g., `alacritty.yml`)
5. **Copy the color section** to your config file
6. **Reload the application** (commands in snippet comments)
7. **Repeat** for other applications

**Example for Alacritty:**
```bash
# View the snippet
cat themes/catppuccin/snippets/alacritty.yml

# Copy colors section to your config
# (manually paste in your editor)

# Reload Alacritty
# No reload needed - automatic on config save
```

### Method 2: Quick Apply Commands

For faster application, use these commands:

```bash
# Alacritty
cat themes/<theme>/snippets/alacritty.yml >> ~/.config/alacritty/alacritty.yml

# Kitty
cat themes/<theme>/snippets/kitty.conf >> ~/.config/kitty/kitty.conf

# i3wm
cat themes/<theme>/snippets/i3-colors.conf >> ~/.config/i3/config
i3-msg reload

# Hyprland
cat themes/<theme>/snippets/hyprland-colors.conf >> ~/.config/hypr/hyprland.conf
hyprctl reload

# Waybar
cat themes/<theme>/snippets/waybar-style.css >> ~/.config/waybar/style.css
killall -SIGUSR2 waybar

# Polybar
cat themes/<theme>/snippets/polybar-colors.ini >> ~/.config/polybar/config.ini
polybar-msg cmd restart

# Rofi
cat themes/<theme>/snippets/rofi.rasi >> ~/.config/rofi/config.rasi

# Dunst
cat themes/<theme>/snippets/dunst-colors.conf >> ~/.config/dunst/dunstrc
killall dunst
```

### Method 3: Example Automation Script

An **optional example script** is provided for reference:

```bash
# View usage
./example-theme-automation.sh

# Apply a theme (with prompts and backups)
./example-theme-automation.sh catppuccin
```

## ğŸ› ï¸ Customization Tips

### Adjusting Opacity

Most configs support transparency:
- **Alacritty**: `window.opacity: 0.95`
- **Kitty**: `background_opacity 0.95`
- **Hyprland**: `decoration { opacity 0.95 }`

### Changing Fonts

All snippets use **JetBrains Mono** (included in `/assets/fonts/`). To change:
- Replace `JetBrains Mono` with your preferred font name
- Adjust `font_size` or `size` values as needed

### Modifying Borders

Default border size is `2px`. To adjust:
- **Hyprland**: `general { border_size 2 }`
- **i3wm**: `default_border pixel 2`
- **Kitty**: `window_border_width 2pt`

### Adjusting Gaps

Default gap size is `10px`. To adjust:
- **Hyprland**: `general { gaps_in 10; gaps_out 20 }`
- **i3wm**: `gaps inner 10; gaps outer 10`

### Corner Rounding

Rounding varies by theme (4-10px). To adjust:
- **Hyprland**: `decoration { rounding 10 }`
- **i3wm**: Not natively supported (use picom)
