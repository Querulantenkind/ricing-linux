# Theme Screenshots

This directory contains screenshots for all themes in the collection.

## Structure

Each theme has its own subdirectory with 4 standard screenshots:

- `overview.png` - Full desktop with multiple windows
- `terminal.png` - Terminal emulator showcase
- `rofi.png` - Application launcher (Rofi) view
- `workflow.png` - Typical workflow with editor and terminal

## Adding Screenshots

When adding screenshots for a theme:

1. Use consistent resolution (1920x1080 recommended)
2. Capture clean, organized desktops
3. Show the theme in realistic use
4. Include variety (terminals, editors, browsers)
5. Compress images to reasonable file sizes

## Screenshot Guidelines

- **Format**: PNG preferred for UI screenshots
- **Resolution**: 1920x1080 or 2560x1440
- **File Size**: Keep under 500KB per screenshot
- **Content**: Show theme colors clearly, avoid personal information
- **Lighting**: Use similar time-of-day/wallpaper style per theme
